## 🧠 Problem Statement

Organizations today assess employees based on **job titles and academic qualifications** — not actual skills. This creates a **"Skills Blind Spot"** where:

- 📉 Skills are constantly evolving — new ones appear, old ones become outdated
- ❓ Skills are ambiguous — similar skills are labeled differently across teams
- 🔍 Skills are invisible — unless explicitly updated or observed
- 📋 Resumes & LinkedIn profiles don't reflect current or accurate skillsets

### 👤 Who Faces This Problem?
- **HRBPs** (HR Business Partners)
- **L&D Managers** (Learning & Development)
- **VP Talent**

---

## 🎯 Design Challenge

> *Design a conceptual experience that empowers HRBPs, L&D Managers, and VP Talent to access clear, actionable, and trustworthy insights about skills management — so they can take upskilling & reskilling decisions more efficiently.*

---

## 💡 My Solution — Skills Intelligence Hub

A centralized **Skills Intelligence Dashboard** that gives HR teams:

```
SEE  →  UNDERSTAND  →  ACT
Real-time     Drill into      Assign
Skills Data   Skill Gaps      Training
```

### ✨ Key Features
- 🗺️ **Skills Heatmap** — Visual overview of team skill strengths and gaps
- 👤 **Employee Skill Profiles** — Individual validated skill cards with proficiency levels
- 🚨 **Gap Alerts** — Instant visibility into critical skill gaps by department
- 📚 **Inline Training Recommendations** — Assign training without switching tools
- ✅ **Validated Skill Badges** — Differentiate assessed skills from self-reported ones

---

## 📐 Design Assumptions

> The assignment allowed reasonable assumptions. Here are mine:

1. HR Manager manages between **50–200 employees**
2. Skills data is collected via **iMocha assessments + self-reporting**
3. The platform is a **desktop-first web application**
4. Users access the dashboard **daily or weekly** for workforce planning
5. Skills are validated through **iMocha's own assessment engine**

## 🔑 Key Design Decisions

| # | Decision | Reasoning |
|---|----------|-----------|
| 01 | **Heatmap for Skills Overview** | Color draws attention 5x faster than text — red zones are immediately visible |
| 02 | **Last Validated Date on Every Skill** | Solves the trust problem — stale data is clearly marked |
| 03 | **Self-Reported vs Assessed Badge** | Not all skills are equal — source of validation matters for decisions |
| 04 | **Inline Training Recommendations** | Reduces tool switching — assign training directly from the gap |
| 05 | **Simple Filters over Complex Search** | HRBPs are not data analysts — reduce cognitive load, speed up decisions |
